3. 在您的 MySQL 服务器中创建一个新的数据库。您可以使用 MySQL 命令行工具或 phpMyAdmin 等图形化工具完成此操作。

5. 将 `maiburogu.sql` 文件导入到您创建的数据库中。该文件包含了所需的所有数据表结构和初始数据。
